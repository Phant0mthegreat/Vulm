# 🛰️ Vulm
Vulm, uma ferramenta criada para otimizar a coleta de informações.

## 🧪 Testado em...
 > - - [X] **KALI LINUX** 

 > - - [x] **TERMUX** 

## 🔧 Instalação no Kali Linux e Termux



 - `git clone https://github.com/Phant0mthegreat/Vulm/`

 - `cd Vulm`
   
 - `bash install.sh`

## 💉 Iniciar
Use o comando ↓
 - `python3 Vulm.py`
para iniciar o programa quando quiser
