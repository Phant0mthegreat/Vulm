# 🛰️ Vulm
Vulm, uma ferramenta criada para otimizar a coleta de informações.

## 🧪 Testado em...
 > - - [ ] **KALI** 

 > - - [x] **TERMUX** 

## 🔧 Instalação no Termux



 - `git clone github.com/Phant0mthegreat/Vulm/`

 - `cd Vulm`
   
 - `pip install -r requirements.txt`

## 💉 Iniciar
Use o comando ↓
 - `python3 Vulm.py`
para iniciar o programa
